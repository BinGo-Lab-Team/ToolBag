# ReICON (Refresh ICON)
删除图标和缩略图缓存并重启资源管理器<br>
此软件作者：**YanAnghelp**

## 额外说明
此脚本理论上仅限于 Windows 7/8/9/10/11，仅在 Windows 10/11 上测试过！<br>
包含简体中文的BAT脚本须使用 **GBK(ANSI)** 编码保存，但 Github 可能会转码转为 UTF-8<br>
为了防止不必要的错误出现，此脚本还有以下机制：<br>

1. 使用 UAC 申请管理员权限(v1.1)
