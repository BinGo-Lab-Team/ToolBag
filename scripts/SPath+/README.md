# SPath+ (System PATH Adder)
将你输入的目录添加到 **系统变量 - PATH** 中<br>
没错，就是这么简单<br>
此软件作者：**YanAnghelp**

## 额外说明
此脚本理论上仅限于 Windows 7/8/9/10/11，仅在 Windows 11 上测试过！<br>
包含简体中文的BAT脚本须使用 **GBK(ANSI)** 编码保存，但 Github 可能会转码转为 UTF-8<br>
为了防止不必要的错误出现，此脚本还有以下机制：<br>

1. 使用 UAC 申请管理员权限
2. 规范化输入路径
3. 检查输入目录的有效性
4. 查重
5. 广播变量变更
